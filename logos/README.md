# Impostor — Logo pack

Sitio: https://impostor.nekix.lol

## Contenido

- `impostor.svg` — vector original
- `favicon.ico` — multi-resolución 16/32/48 px
- `favicon-16x16.png` · `favicon-32x32.png` · `favicon-48x48.png`
- `apple-touch-icon.png` — 180 × 180 (iOS)
- `android-chrome-192x192.png` · `android-chrome-512x512.png` (PWA)
- `logo-256x256.png` · `logo-1024x1024.png`
- `site.webmanifest` — para PWA

## Instalación rápida

Copiá todo a `public/` y agregá al `<head>`:

```html
<link rel="icon" type="image/x-icon" href="/favicon.ico">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png">
<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png">
<link rel="manifest" href="/site.webmanifest">
```
